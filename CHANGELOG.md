# Changelog

All notable changes to Yano Space are documented here. Format follows
[Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and versioning
follows [Semantic Versioning](https://semver.org/) adapted for a pre-1.0
beta:

- **PATCH** (0.x.y): bug fixes only, no behavior/feature changes
- **MINOR** (0.x.0): new features or notable user-facing changes
- **MAJOR** (1.0.0+): first stable release, then breaking changes only

## [0.1.0] - 2026-07-17

### Added
- First public beta build. One-knob space processor for amapiano
  production: high-shelf air EQ, spacious stereo reverb, and gentle M/S
  stereo widening (bass kept mono below the width crossover), all driven
  by a single Amount macro.
- Airier, more reverberant character than Montagem Widener's pure M/S
  width, researched against amapiano production references.
- Builds as VST3, AU (passes `auval`), and Standalone on macOS and Windows.
